# 📘 [Parametros y argumentos en los metodos]

## 🧠 Qué aprendí (máx 3 puntos)
el parametro se crea cuando se esta creando el metodo osea se agregan en la firma del metodo

el argumento son los valores que se pasan en parentesis cuando se llama al metodo

los argumentos son recepcionados por el metodo en el mismo orden que fueron declarados en la firma

---

## ⚙️ Cómo funciona (explícalo con tus palabras)
los parametros se crean en el metodo para cualquier uso que se necesite, aveces retorna algo y otras veces no

---

## 💻 Ejemplo práctico

aqui se crean los parametros que son int a y int b
public static int sumar(int a, int b) {
        int resultado = a + b;
        return resultado;
    }

aca se pasan los argumentos que son los dos enteros
int resultado = sumar(4, 3);
